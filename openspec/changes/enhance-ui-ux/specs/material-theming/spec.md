# Material Theming

## ADDED Requirements

### Requirement: Material Design Styling
The application MUST use Material Design 3 styling principles (colors, spacing, elevation).

#### Scenario: Visual Consistency
- Given the application is loaded
- When the user views buttons and cards
- Then they exhibit MD3 properties (rounded corners, elevation shadows, ripple effects).

### Requirement: Configurable Theme
The application MUST allow the user to select a primary color theme.

#### Scenario: Change Theme
- Given the Settings menu is open
- When the user selects "Purple"
- Then the application's primary color changes to Purple immediately.
- And the preference is saved.
